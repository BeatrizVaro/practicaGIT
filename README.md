## Proceso del GIT
1.Crear repositorio
>Ir a [Github](https://github.com)
2.Hacer el clone
```console
git clone https://github.com/BeatrizVaro/practicaGIT.git
```
3.Hacemos cambios
4.Guardamos con un commit
```console
git status
git add <nombre archivo>
git commit -m "Mensaje"
```
5.Hacemos el push
```console
git push
```
6. Tras otros cambios bajamos con un pull
```console
git pull
```
